# To-Do List Application  

A simple and interactive **To-Do List Application** built using **Python (Tkinter)**.  
This project provides a GUI-based task management tool where users can add, update, and delete tasks easily.  

---

## Features  

-  Add new tasks.  
-  Delete a single task or **delete all** tasks.  
-  Sort tasks in **ascending or descending order**.  
-  Choose a **random task**.  
-  Display total **number of tasks**.  
-  Simple and user-friendly interface.  

---

## Tech Stack  

- **Python 3.12.3 
- **Tkinter** (for GUI)  

---

## Getting Started  

### 1. Clone the Repository  
```bash
git clone https://github.com/your-username/to-do-list.git
cd to-do-list
```

### 2. Run the Application  
Make sure you have **Python 3.x** installed, then run:  
```bash
python todo_list.py
```

---



##  How to Use  

1. Launch the application.  
2. Enter a task in the input field and click **Add Task**.  
3. Use **delete** to remove one task or **delete all** to clear the list.  
4. Sort tasks using **Ascending / Descending**.  
5. Use **choose random** to pick a random task.  
6. Check **number of tasks** to know your list size.
